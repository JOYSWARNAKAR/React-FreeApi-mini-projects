# YouTube Videos Listing UI

A simple YouTube-style video listing app built using React + Vite.

## Features

- Fetch videos from FreeAPI
- Display videos in card layout
- Responsive UI
- Clean design

## Project Structure
```
src/
├── components/
│   └── VideoCard.jsx
├── App.jsx
├── main.jsx
└── index.css
```
