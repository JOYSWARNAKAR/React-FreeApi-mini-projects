# Product Listing Interface

A simple Product Listing App built using React + Vite.

## Features

- Fetch products from FreeAPI
- Display products in card layout
- Responsive UI
- Clean and simple design

## Project Structure
```
src/
├── components/
│   └── ProductCard.jsx
├── App.jsx
├── main.jsx
└── index.css
```
