# Random Users UI

A simple Random Users App built using React + Vite.

## Features

- Fetch users from FreeAPI
- Display users in profile cards
- Responsive UI
- Modern dark design

## Project Structure

```
src/
├── components/
│   └── UserCard.jsx
├── App.jsx
├── main.jsx
└── index.css
```
