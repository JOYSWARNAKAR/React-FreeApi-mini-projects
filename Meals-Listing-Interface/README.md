# Meals Listing Interface

A simple Meals Listing App built using React + Vite.

## Features

- Fetch meals from FreeAPI
- Display meals in card layout
- Responsive UI
- Modern dark design

## Project Structure
```
src/
├── components/
│   └── MealCard.jsx
├── App.jsx
├── main.jsx
└── index.css
```
