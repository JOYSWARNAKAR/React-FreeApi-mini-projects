# Random Cat Viewer

A simple Random Cat Viewer app built using React + Vite.

## Features

- Fetch random cat images from FreeAPI
- Display cat images dynamically
- Refresh image with button click
- Simple and clean UI

## Project Structure
```
src/
├── App.jsx
├── main.jsx
└── index.css
```
