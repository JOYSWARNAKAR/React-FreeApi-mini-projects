# Quotes Listing Application

A simple Quotes Listing App built using React + Vite.

## Features

- Fetch quotes from FreeAPI
- Display quotes in card layout
- Responsive UI
- Simple and clean design

## Project Structure
```
src/
├── components/
│   └── QuoteCard.jsx
├── App.jsx
├── main.jsx
└── index.css
```
